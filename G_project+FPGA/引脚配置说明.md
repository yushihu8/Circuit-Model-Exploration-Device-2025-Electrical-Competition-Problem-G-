# G_Project 工程引脚配置说明

## 工程信息
- **芯片型号**: xc7z020clg484-1 (Zynq-7020)
- **工作电压**: LVCMOS33 (3.3V)

## 时钟约束

### 主时钟
```tcl
create_clock -period 20.000 -name clk -waveform {0.000 10.000} [get_ports clk]
```
- 频率: 50MHz
- 周期: 20ns

### 内部生成时钟 (40.96MHz)
```tcl
create_clock -period 24.414 -name clk_40_96m -waveform {0.000 12.207} [get_nets clk_wiz_40_96m_inst0/clk_out1]
```
- 频率: 40.96MHz
- 周期: 24.414ns
- 用于ADC/DAC数据处理

## 电气特性总结

| 参数 | 配置 | 适用信号 |
|------|------|---------|
| 电压标准 | LVCMOS33 | 所有信号 |
| ADC时钟转换速率 | SLOW | adc_clk0, adc_clk1 |
| ADC时钟驱动强度 | 12mA | adc_clk0 |
| DAC时钟转换速率 | FAST | dac_clk |
| DAC数据转换速率 | FAST | dac_out[13:0] |
| DAC驱动强度 | 12mA | dac_out[13:0] |
| SPI MISO驱动强度 | 12mA | spi_miso |
| SPI MISO转换速率 | SLOW | spi_miso |
| 输出端接 | NONE | adc_clk0, adc_clk1, dac_clk, dac_out[13:0], spi_miso |

## ILA调试配置

```tcl
set_property C_CLK_INPUT_FREQ_HZ 300000000 [get_debug_cores dbg_hub]
set_property C_ENABLE_CLK_DIVIDER false [get_debug_cores dbg_hub]
set_property C_USER_SCAN_CHAIN 1 [get_debug_cores dbg_hub]
connect_debug_port dbg_hub/clk [get_nets clk_out40_96m]
```

调试时钟连接到40.96MHz内部时钟。

---

## 注意事项

1. **ADC采样率**: 约204.8kHz (40.96MHz / 200)
2. **DAC时钟**: 使用40.96MHz时钟反相 (~clk_out40_96m)
3. **IIR滤波器**: 工程包含IIR数字滤波器模块
4. **双路ADC**: 支持两路独立14位ADC采集
5. **SPI控制**: 通过SPI接口配置滤波器系数和读取ADC数据
